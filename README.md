# GvSync 1.1.12

**GvSync** is a simple GUI-based tool to toggle NVIDIA G-SYNC on or off.

## Features
- Toggle G-SYNC **ON** or **OFF** with a click.
- Displays current G-SYNC status.
- Stay in the system tray for easy acces.
- Minimalistic and stylish dark mode interface.

## Updates
- Console log fixed
- GvSync is now in the system tray ( Close the app here )

## Knows issues
- Command prompt may appear for a few seconds at the start.
- The status is not updated on first use when registry is not created.
- Settings is not completed.
